- ^：异或	相同则为0，不同则为1。异或是一类逆运算

```bash
^是^逆运算, ^与自己互为逆运算
a ^ b = c
=> c ^ b = a, c ^ a = b
由^ 异或运算方式演算而来，相同则为0，不同则为1
两个相同的值做异或运算的时候为0
2^2=0
0^2=2,0^n=n
------------------------------------------------
逆运算：有一种前提
当前这种运算满不满足交换律
a+b  b+a
a^b  b^a
结果没区别，一样
```
